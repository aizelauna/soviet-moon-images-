Generic Flammable Object (#2002)

(an instance of generic thing made by Trismegistos) 




Burning noises:


      %D snaps, crackles, and pops in the flames


      Flames and black smoke spiral upwards from %D




Go to location of this object, AnArchie Bunker.






HELP MANUAL:



     Generic Flammable Object is intended for use with Trism's Gasoline Can. It
     contains no verbs itself, but has useful properties that will help you
     custom-create bonfires, effigies, and even custom-engineer your own fiery
     "suicide."
     
     Message properties:
     
     @ignite  is 
     @burndesc  is 
     @burnout  is 
     
     List property:
     .burn_noise: a list of phrases randomly announced one a minute as the
     object burns. Use @notedit to edit, with each phrase on a separate line.
     
     "Physical" properties:
     
     .burn_minutes: logically enough, the number of minutes the object will burn. 
Default is 1 minute, plus 1 minute per Kbyte of object size.
     .destructible: if 0, the object will not be recycled when it is done burning. 
Default is 1.
     
     NOTE: To make yourself a customized human torch, create these properties on yourself. 
Don't forget, the message properties are in the format .ignite_msg , etc. .destructible 
will not have any effect on a self-immolating player.
