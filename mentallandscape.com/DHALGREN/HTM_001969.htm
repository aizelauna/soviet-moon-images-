How to Send Mail (#1969)

(an instance of generic thing made by Wizard) 

     Creating a note costs 10 pennies. @create note from MyName to Whoever; @describe 
note=; @lock note=me | *whoever; whoever-box; drop note. If you want your note to 
be private, @describe note=It's a private note to Whoever. @success note=; @fail 
note=It's not for you.. If you want your note to return home when Whoever drops it, 
@set note=sticky (after you drop it).
