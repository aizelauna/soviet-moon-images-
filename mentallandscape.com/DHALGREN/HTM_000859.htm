Generic Moon-phased Wearable (#859)

     A wearable thing, whose description will change with the moon for lycanthrope pc's (#1223)...

Go to location of this object, wooden box.


HELP MANUAL: 
     This is a wearable object, but with a twist...
     Markup strings It's a crescent moon, and  waxing. may be included in its @worn message. 
If the player wearing the object is a child of #1223 (the generic lycanthrope), messages 
from the set @full_moon, @new_moon, @gibbous_moon, @half_moon, @crescent_moon, @moon_waxing 
and @moon_waning will be substituted for these strings according to the present phase 
of the moon, thus phasing the @worn message. See 'help messages' for help on how 
to set these.
     The moon, for players who don't know, is #932 in Dhalgren...(its phases correspond 
to actual phases of the earth's largest satellite).
     If the player wearing the object is not a descendent of the lycanthrope (#1223), 
the object will look kind of funny.
     For more help on marking up descriptions (or in this case the @worn message), see 
#110, the Post-Apocalyptic Room, which performs similar substitutions for weather 
and daytime...
